# ポートフォリオ概要
石油暖房機メーカーの販売データーを「年代別の年齢」を切り口で可視化するデモダッシュボードを作成しました。
デモダッシュボードで使用させて頂いたメーカーは1社目でお世話になった企業です。
エクセルで、RANDARRAY関数を用いてダミーデーターを作成しています。

# 使用環境
Salesforce の無料で使える開発環境「Developer Edition」


# Login
